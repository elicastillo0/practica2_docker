🎓 Práctica 2 – Docker Compose: API Node.js con MongoDB

En esta práctica se construye una API REST con Node.js + Express conectada a una base de datos MongoDB, todo orquestado mediante Docker Compose. El objetivo es levantar una arquitectura completa con servicios independientes que se comunican entre sí dentro de un entorno Dockerizado.

🚀 Objetivo de la práctica

Crear una API con Express que gestione equipos de la NBA.

Conectar la API a una base de datos MongoDB.

Definir servicios en Docker Compose para levantar ambos contenedores.

Validar la conexión mediante variables de entorno (MONGO_URI).

Exponer endpoints funcionales para consultar datos.
